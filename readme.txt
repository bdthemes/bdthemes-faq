== Description ==

**BdThemes FAQ** is a powerful and lightweight WordPress plugin that adds a custom FAQ post type to your website and provides a modern Elementor widget to display your FAQs in a beautiful list layout.

= KEY FEATURES =

* **Custom FAQ Post Type** - Dedicated post type for managing FAQs
* **FAQ Categories (Filters)** - Organize FAQs with custom taxonomy
* **Elementor Widget** - Beautiful FAQ List widget for Elementor page builder
* **Featured Image Support** - Display featured images or custom icons with each FAQ
* **Excerpt Support** - Show short descriptions for each FAQ
* **Category Display** - Show FAQ categories as styled badges
* **Flexible Ordering** - Sort by date, title, menu order, or random
* **Filter by Category** - Display FAQs from specific categories
* **Fully Customizable** - Complete control over styling with Elementor
* **Responsive Design** - Mobile-friendly and looks great on all devices
* **Clean Code** - Follows WordPress coding standards
* **Translation Ready** - Fully translatable with .pot file included
* **GPL3 Licensed** - 100% free and open source

= ELEMENTOR WIDGET FEATURES =

The FAQ List widget provides extensive customization options:

== Installation ==

= Minimum Requirements =

* WordPress 5.0 or greater
* PHP version 7.4 or greater
* MySQL version 5.0 or greater
* Elementor 3.28.0 or greater

= Automatic Installation =

1. Go to your WordPress Dashboard
2. Navigate to Plugins > Add New
3. Search for "BdThemes FAQ"
4. Click "Install Now" and then "Activate"

= Manual Installation =

1. Download the plugin ZIP file
2. Go to WordPress Dashboard > Plugins > Add New
3. Click "Upload Plugin" button
4. Choose the downloaded ZIP file and click "Install Now"
5. After installation, click "Activate Plugin"

= Using the Plugin =

After activation:

1. Go to Dashboard > FAQ to create your first FAQ item
2. Add title, description, featured image (optional), and custom icon (optional)
3. Assign categories using FAQ Filter taxonomy
4. Edit any page with Elementor
5. Search for "FAQ List" widget and drag it to your page
6. Configure the widget settings as needed
7. Style it using the extensive customization options
8. Save and preview your page

== Frequently Asked Questions ==

= Does this plugin require Elementor? =

Yes, this plugin requires Elementor page builder (free version is sufficient) to display the FAQ List widget. However, the custom FAQ post type will work without Elementor.

= Can I use custom icons with FAQs? =

Yes! Each FAQ can have a custom icon/image that you can upload through the meta box in the FAQ editor.

= Can I use featured images instead of icons? =

Absolutely! The widget allows you to choose between displaying the featured image or custom icon for each FAQ.

= Can I filter FAQs by category? =

Yes, you can filter FAQs by categories using the widget settings. You can select one or multiple categories to display.

= Is it translation ready? =

Yes, the plugin is fully translation ready with .pot file included in the languages folder.

= Can I customize the design? =

Yes! The Elementor widget provides extensive styling options including colors, typography, spacing, borders, shadows, and hover effects.

= Does it work with any WordPress theme? =

Yes, BdThemes FAQ works with any WordPress theme that supports Elementor.

= Is it mobile responsive? =

Yes, the FAQ display is fully responsive and works perfectly on all devices.

= Can I change the permalink structure? =

Yes, you can customize the FAQ slug through WordPress Customizer (if your theme supports it) or by modifying the plugin settings.

= Do you offer support? =

Yes, we provide support through the WordPress.org support forum. For premium support, please visit our website.
